# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly through [GitHub Security Advisories](https://github.com/williamzujkowski/tsundoku/security/advisories/new).

Do not open a public issue for security vulnerabilities.

## Scope

This is a static site generator for a book collection. The primary security concerns are:

- Supply chain attacks via dependencies
- Build-time code injection
- Cross-site scripting in generated output

## Response

I will acknowledge reports within 72 hours and provide a fix timeline within one week.
